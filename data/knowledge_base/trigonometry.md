# Trigonometry (Comprehensive - JEE Scope)

## Basic Ratios & Identities
- $\sin^2\theta + \cos^2\theta = 1$
- $1 + \tan^2\theta = \sec^2\theta$
- $1 + \cot^2\theta = \csc^2\theta$
- $\sin\theta \cdot \csc\theta = 1$, $\cos\theta \cdot \sec\theta = 1$, $\tan\theta \cdot \cot\theta = 1$

## Allied Angle Formulas
- $\sin(-\theta) = -\sin\theta$, $\cos(-\theta) = \cos\theta$
- $\sin(90° - \theta) = \cos\theta$, $\cos(90° - \theta) = \sin\theta$
- $\sin(90° + \theta) = \cos\theta$, $\cos(90° + \theta) = -\sin\theta$
- $\sin(180° - \theta) = \sin\theta$, $\cos(180° - \theta) = -\cos\theta$
- $\sin(180° + \theta) = -\sin\theta$, $\cos(180° + \theta) = -\cos\theta$

## Compound Angle Formulas
- $\sin(A \pm B) = \sin A \cos B \pm \cos A \sin B$
- $\cos(A \pm B) = \cos A \cos B \mp \sin A \sin B$
- $\tan(A \pm B) = \dfrac{\tan A \pm \tan B}{1 \mp \tan A \tan B}$

## Double Angle Formulas
- $\sin 2A = 2\sin A \cos A = \dfrac{2\tan A}{1 + \tan^2 A}$
- $\cos 2A = \cos^2 A - \sin^2 A = 1 - 2\sin^2 A = 2\cos^2 A - 1 = \dfrac{1 - \tan^2 A}{1 + \tan^2 A}$
- $\tan 2A = \dfrac{2\tan A}{1 - \tan^2 A}$

## Triple Angle Formulas
- $\sin 3A = 3\sin A - 4\sin^3 A$
- $\cos 3A = 4\cos^3 A - 3\cos A$
- $\tan 3A = \dfrac{3\tan A - \tan^3 A}{1 - 3\tan^2 A}$

## Half Angle Formulas
- $\sin\frac{A}{2} = \pm\sqrt{\dfrac{1 - \cos A}{2}}$
- $\cos\frac{A}{2} = \pm\sqrt{\dfrac{1 + \cos A}{2}}$
- $\tan\frac{A}{2} = \pm\sqrt{\dfrac{1 - \cos A}{1 + \cos A}} = \dfrac{\sin A}{1 + \cos A} = \dfrac{1 - \cos A}{\sin A}$

## Sum-to-Product Formulas
- $\sin C + \sin D = 2\sin\dfrac{C+D}{2}\cos\dfrac{C-D}{2}$
- $\sin C - \sin D = 2\cos\dfrac{C+D}{2}\sin\dfrac{C-D}{2}$
- $\cos C + \cos D = 2\cos\dfrac{C+D}{2}\cos\dfrac{C-D}{2}$
- $\cos C - \cos D = -2\sin\dfrac{C+D}{2}\sin\dfrac{C-D}{2}$

## Product-to-Sum Formulas
- $2\sin A \cos B = \sin(A+B) + \sin(A-B)$
- $2\cos A \sin B = \sin(A+B) - \sin(A-B)$
- $2\cos A \cos B = \cos(A+B) + \cos(A-B)$
- $2\sin A \sin B = \cos(A-B) - \cos(A+B)$

## Properties of Triangle
For a triangle with sides $a, b, c$ and opposite angles $A, B, C$:
- **Sine Rule**: $\dfrac{a}{\sin A} = \dfrac{b}{\sin B} = \dfrac{c}{\sin C} = 2R$ (where $R$ = circumradius)
- **Cosine Rule**: $\cos A = \dfrac{b^2 + c^2 - a^2}{2bc}$
- **Projection Formula**: $a = b\cos C + c\cos B$
- **Area of Triangle**: $\Delta = \dfrac{1}{2}ab\sin C$
- **Hero's Formula**: $\Delta = \sqrt{s(s-a)(s-b)(s-c)}$ where $s = \dfrac{a+b+c}{2}$
- **Inradius**: $r = \dfrac{\Delta}{s}$
- **Circumradius**: $R = \dfrac{abc}{4\Delta}$

## Inverse Trigonometric Functions
### Domain and Range
| Function | Domain | Range |
|---|---|---|
| $\sin^{-1} x$ | $[-1, 1]$ | $[-\pi/2, \pi/2]$ |
| $\cos^{-1} x$ | $[-1, 1]$ | $[0, \pi]$ |
| $\tan^{-1} x$ | $\mathbb{R}$ | $(-\pi/2, \pi/2)$ |

### Key Identities
- $\sin^{-1} x + \cos^{-1} x = \dfrac{\pi}{2}$, $\quad |x| \le 1$
- $\tan^{-1} x + \cot^{-1} x = \dfrac{\pi}{2}$
- $\sin^{-1}(-x) = -\sin^{-1} x$, $\cos^{-1}(-x) = \pi - \cos^{-1} x$
- $\tan^{-1}(-x) = -\tan^{-1} x$
- $\tan^{-1} x + \tan^{-1} y = \tan^{-1}\dfrac{x+y}{1-xy}$, when $xy < 1$
- $2\tan^{-1} x = \sin^{-1}\dfrac{2x}{1+x^2} = \cos^{-1}\dfrac{1-x^2}{1+x^2} = \tan^{-1}\dfrac{2x}{1-x^2}$

## General Solutions of Trigonometric Equations
- $\sin\theta = 0 \Rightarrow \theta = n\pi$
- $\cos\theta = 0 \Rightarrow \theta = (2n+1)\dfrac{\pi}{2}$
- $\tan\theta = 0 \Rightarrow \theta = n\pi$
- $\sin\theta = \sin\alpha \Rightarrow \theta = n\pi + (-1)^n \alpha$
- $\cos\theta = \cos\alpha \Rightarrow \theta = 2n\pi \pm \alpha$
- $\tan\theta = \tan\alpha \Rightarrow \theta = n\pi + \alpha$
